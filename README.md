# 🏆 Prode Mundial 2026

Aplicación web para organizar un prode privado del Mundial FIFA 2026 entre amigos.

Los participantes pueden registrarse, cargar sus pronósticos para cada partido, seguir la tabla de posiciones en tiempo real y conversar mediante un chat integrado.

---

## ✨ Características

- ⚽ Pronósticos para todos los partidos del Mundial 2026.
- 🔐 Registro e inicio de sesión con contraseña.
- 🏅 Tabla de posiciones actualizada automáticamente.
- 💬 Chat grupal integrado.
- 🎭 Avatares personalizables.
- 📊 Sistema de puntaje simple y transparente.
- ☁️ Sincronización en tiempo real mediante Supabase.
- 🔧 Panel de administración para cargar resultados oficiales.
- 📱 Diseño optimizado para celulares.

---

## 🎯 Sistema de puntuación

| Acción | Puntos |
|----------|----------|
| Acertar ganador o empate | 3 |
| Acertar goles del local | 1 |
| Acertar goles del visitante | 1 |

**Máximo posible: 5 puntos por partido.**

### Ejemplos

| Resultado real | Pronóstico | Puntos |
|---|---|---|
| 2–1 | 2–1 | 5 |
| 2–1 | 2–0 | 4 |
| 2–1 | 1–0 | 3 |
| 2–1 | 1–1 | 0 |

---

## 🎮 Cómo se juega

1. Crear una cuenta.
2. Elegir un avatar.
3. Configurar un resultado por defecto.
4. Cargar los pronósticos antes del comienzo de cada partido.
5. Seguir la tabla de posiciones en tiempo real.
6. Competir contra tus amigos.
7. Comentar los resultados en el chat integrado.
